Habanero-UPC++ Overview

The different types of supported intra-place asynchronous tasks are as following: 

async([capture_list]() {
  S1;
});

// async with Data-Driven Futures (DDFs)
asyncAwait(ddf_1, ddf_2, ...., [capture_list]() {
  S2;
});

// async with Phasers
asyncPhased(ph_1, mode_1, ph_2, mode_2, ...., [capture_list]() {
  S3;
});

// parallel for loop
forasync(dim, style, ind1, ind2, ..., siz1, siz2, ..., 
			     seq1, seq2, ..., [capture_list]() {
  S4;
});

The different types of supported inter-place asynchronous tasks (extended from UPC++) are as following: 

// asynchronous task invocation at remote place
asyncAt(place, [capture_list]() {
  S5;
});

// asynchronous remote copy
asyncCopy(global_ptr src, global_ptr dst, size_t count, DDF* ddf=NULL);

Both intra-place and inter-place asynchronous tasks can be joined by using a special finish: 

finish_spmd([capture_list]() {
  async(....);
  asyncAwait(...);
  asyncPhased(...);
  forasync(...);
  asyncAt(...);
  asyncCopy(...);
});

A normal finish can be used for joining only intra-place asynchronous tasks. Tasks in this finish scope cannot spawn any inter-place asynchronous tasks.

finish([capture_list]() {
  async(....);
  asyncAwait(...);
  asyncPhased(...);
  forasync(...);
});
